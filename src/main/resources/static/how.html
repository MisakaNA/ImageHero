How to get pixiv id and login cookie?

For pixiv id:

1. Go to Pixiv, select an artwork you like

2. Find the numeric id number at the end of the url for the web page:

 

3. After you logged in on the Pixiv site, in the right click menu and select "inspect"

to open the inspection window, and then select "Network":

and refresh the page

4. Select "All" and Put "illust?mode=all&lang=" in the "Filter":

5. Click any one of the links in the list to open the newwork package, and under "Headers" tab, scroll down until you find an entry

called "cookie", and value is start with "first_visit_datetime" or "p_ab_id":

copy the whole value and paste it to the login cookie
